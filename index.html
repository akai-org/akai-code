AKAI Code

Pierwsza edycja Hackatonu AKAI Code organizowana przez AKAI - Akademickie Koło Aplikacji Internetowych - odbędzie się na Politechnice Poznańskiej

Założenie wydarzenia jest dziecinnie proste! Kodujesz przez 24 godziny aplikację nawiązującą do tematu, a następnie jury decydują, która drużyna wygra.

Konkurować można w trzech kategoriach - Najlepszy pomysł, najlepszy prototyp i najlepsza zastosowana technologia (innowacyjność). Jaki będzie temat? Tego dowiecie się już wkrótce!
